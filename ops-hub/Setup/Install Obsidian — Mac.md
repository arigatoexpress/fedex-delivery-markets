# 🍎 Install Obsidian — Mac

About 5 minutes. Works on both Apple Silicon (M1/M2/M3/M4) and Intel Macs.

## Steps
1. Go to **https://obsidian.md/download** in Safari or Chrome.
2. Click **Get Obsidian for macOS**. A file like `Obsidian-x.x.x.dmg` downloads.
3. Open the downloaded `.dmg` file (double-click it).
4. In the window that appears, **drag the Obsidian icon onto the Applications
   folder.**
5. Open **Applications** and double-click **Obsidian**.
   - If macOS says *"Obsidian can't be opened because it is from an unidentified
     developer,"* right-click the app → **Open** → **Open** again. (Only needed
     the first time.)
6. On the welcome screen, click **Open folder as vault**.
7. Select this folder: **`FedEx-Delivery-Markets-Ops-Vault`**
   - If your team syncs it via Google Drive or iCloud/OneDrive, browse to that
     location. See [[Sync with Google Drive]] or [[Sync with OneDrive (Microsoft)]].
8. If Obsidian asks **"Trust author and enable plugins?"**, click
   **Trust author and enable plugins**.
9. Open **`00 — START HERE`** in the left sidebar. 🎉
