# 🪟 Install Obsidian — Windows

About 5 minutes. No admin rights needed for the standard installer.

## Steps
1. Go to **https://obsidian.md/download** in your browser.
2. Click **Get Obsidian for Windows**. A file like `Obsidian-x.x.x.exe` downloads.
3. Open the downloaded file. If Windows shows a blue "Windows protected your PC"
   box, click **More info → Run anyway** (this is normal for new downloads).
4. Obsidian installs and opens on its own.
5. On the Obsidian welcome screen, click **Open folder as vault**.
6. Browse to and select this folder: **`FedEx-Delivery-Markets-Ops-Vault`**
   - If your team keeps it in OneDrive, it's usually under
     `C:\Users\<you>\OneDrive\...`. See [[Sync with OneDrive (Microsoft)]].
7. If Obsidian asks **"Trust author and enable plugins?"**, click
   **Trust author and enable plugins** (this hub only uses the Dataview add-on).
8. Open the note **`00 — START HERE`** in the left sidebar. 🎉
