# 🗺️ Visual maps

The fastest way to "get" this project is to look at it. Three visual boards:

| Map | What it shows | Best for |
|---|---|---|
| [[System Overview.canvas\|🧭 System Overview (Canvas)]] | The whole demo flow on one screen, with clickable cards into the guides | Understanding how the app works |
| [[Delivery Route Map.excalidraw\|🚚 Delivery Route Map]] | A package's journey + where markets **open / lock / resolve** along the route | Explaining the *concept* to someone |
| [[Last-Mile Route Map.excalidraw\|🚐 Last-Mile Route Map]] | A driver's stops with a YES/NO "on-time" market per stop (illustrative) | Showing *why* a delivery market is useful |

> [!tip] The route maps are **Excalidraw** drawings — you can edit them like a
> whiteboard (drag boxes, recolor, add notes). Click one, then use the **More
> options (•••) → Switch to Excalidraw view** if it opens as text.

> [!warning] Everything here is **paper-only / illustrative**. The route maps use
> made-up examples — see [[Safety & What This Is NOT]].
