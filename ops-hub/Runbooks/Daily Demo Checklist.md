# ✅ Daily Demo Checklist

Print this or keep it open before a demo. Tick the boxes.

## Before the meeting
- [ ] I've read [[Safety & What This Is NOT]] recently.
- [ ] The app is started and I can open it in a browser
      (`http://127.0.0.1:5178` for dev, or `http://127.0.0.1:4747` for the build).
- [ ] I have the [[Demo Tracking Numbers]] handy (or this hub open).
- [ ] I tried each demo number once so I know what each screen looks like.
- [ ] I know my opening and closing "safety frame" lines.

## During the demo
- [ ] I said the **opening frame** ("paper-only, fake data, no real money").
- [ ] I showed an **open** market (`771234567890`).
- [ ] I showed a **quote** and a **paper order** (stressed the word *paper*).
- [ ] I showed a **resolved** market (`993456789012`).
- [ ] I said the **closing frame**.

## After the meeting
- [ ] I filled out a **[[Templates/Demo Session Log|Demo Session Log]]**.
- [ ] I added any new questions to **[[Decisions & Open Questions]]**.
- [ ] If something broke → noted in [[Troubleshooting]] / told the owner.
- [ ] If anything looked unsafe → followed [[Incident & Escalation]].
